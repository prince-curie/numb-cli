# numb-cli

A command line application that returns a random fact based on the number you input and the category you select.

## INSTALLATION

- `git clone https://github.com/prince-curie/numb-cli.git` 
- `cd numb-cli`
- `tsc`
- `npm link`

## USAGE

### To View Commands
`numb-cli`

### To Get a Random Fact
`numb-cli random`
